# 🔢 Tiering System

MercurX tiering system is designed for MercurX community to fair access to projects. There are three different tiers in MercurX:

 

### Tier 1: 

Requirement: Staked minimum 1000 MERX

First Come First Served

Access to private sale if Private Sale Pass activated

  

### Tier 2:

Requirement: Staked 4000 MERX

Guaranteed allocation size for projects: Medium

Access to private sale if Private Sale Pass activated

  

### Tier 3:

Requirement: Staked 8000 MERX

Guaranteed allocation size for projects: High

Access to private sale

Voting rights

Private sale pass
